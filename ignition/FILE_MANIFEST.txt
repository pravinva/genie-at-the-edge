QUICK REFERENCE - DATABASE QUERIES
================================================================================

Core Queries:
1. approveRecommendation
   UPDATE agent_recommendations
   SET status='approved', operator_id=:op_id, approved_timestamp=NOW()
   WHERE recommendation_id=:rec_id

2. deferRecommendation
   UPDATE agent_recommendations
   SET status='deferred', defer_until=NOW()+INTERVAL '1 HOUR'
   WHERE recommendation_id=:rec_id

3. rejectRecommendation
   UPDATE agent_recommendations
   SET status='rejected', rejection_reason=:reason, rejected_timestamp=NOW()
   WHERE recommendation_id=:rec_id

4. countPendingRecommendations
   SELECT COUNT(*) as count FROM agent_recommendations WHERE status='pending'

Data Queries:
5. Get Pending Recommendations
   SELECT * FROM agent_recommendations
   WHERE status='pending'
   ORDER BY priority DESC, created_timestamp DESC
   LIMIT 50

6. Get Executing Commands
   SELECT * FROM agent_commands
   WHERE status IN ('pending', 'executing')
   ORDER BY status DESC, created_timestamp DESC
   LIMIT 100
